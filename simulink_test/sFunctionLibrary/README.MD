## READ ME FOR S FUNCTIONS ##
# Folder Directory
sFunctionLibrary.slx: Library of s-function blocks that link to the 
source code.

/mexFiles: Contains the compiled binaries corresponding to the s-function 
blocks in the sFunctionLibrary.slx.

*_Harness.slx: Test harness for a particular block within the sFunctionLibrary.slx
DO NOT EDIT, these are autogenerated files.

/testFunctions: This folder contains the functions necessary to run the test harnesses.
run "RunTests.m" from inside the testFunctions folder to execute all 
"*_Test.m" unit tests within the folder.
